## How to Check Live Stats

### Via npm
```bash
npm info @portel/ncp
# Or use npm-stat for detailed analytics
npx npm-stat @portel/ncp
```

### Via GitHub API
```bash
# Run our custom script
node scripts/check-mcpb-downloads.js
```

### Via Shields.io API
```bash
# npm downloads (all time)
curl https://img.shields.io/npm/dt/@portel/ncp.json

# GitHub release downloads (all time)
curl https://img.shields.io/github/downloads/portel-dev/ncp/total.json
```
